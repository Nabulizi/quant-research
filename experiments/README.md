# Experiments

Each experiment gets one directory named `QR-NNN-short-name` after its
preregistration is committed.

An experiment directory contains only the implementation and small configuration
needed to reproduce that registered test. Do not overwrite a completed experiment
to test a changed signal, universe, breadth, cost model, or decision rule. Assign a
new ID and explain the relationship in both preregistrations.

Historical `QR-001` through `QR-007` algorithms remain in `../should-i-trade` and
are not duplicated here.
