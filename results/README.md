# Results

Create one directory per experiment using the same `QR-NNN-short-name` as its
implementation. Commit compact evidence that another researcher can inspect:

- `summary.md` with the frozen criterion and PASS, FAIL, or INVALID verdict.
- `manifest.json` with repository commit, QC project/backtest ID, dates, costs,
  universe rules, breadths, and random seed base.
- Small machine-readable metric tables where useful.

Do not commit credentials, proprietary raw QC data, or large raw logs. Record the
QC identifier and a checksum or concise extraction instead.
